Join the CUE community

CUE is an open source project that anyone in the community can use, improve, and enjoy. We'd love you to join us! Here's a few ways to find out what's happening and get involved.

Learn and Connect

Using or want to use CUE? Find out more here:

  • GitHub Discussions: Ask questions and discuss CUE
  • Slack: Chat with other CUE enthusiasts

Develop and Contribute

If you want to get more involved by contributing to CUE, join us here:

  • GitHub: Issue and proposal tracking takes place here!
  • Gerrit: The source-of-truth repo.

You can find out how to contribute to these docs in our Contribution Guidelines.
